# Star Wars Planets API

Desafio B2W.

Funcionalidades: 
- Adicionar um planeta (com nome, clima e terreno)
- Listar planetas
- Buscar por nome
- Buscar por ID
- Remover planeta

# Run

- Run `npm install` 
- Run `mongod` 
- Run `npm run start`
